## ✨ Features

- 🛍️ Product catalog with category filters (Running, Lifestyle, Urban, Trail)
- 👟 Size selector for each product
- 🛒 Persistent cart using localStorage (survives page refresh)
- 🔐 Login & Signup with form validation
- 💳 Razorpay-style payment UI (Card, UPI, Netbanking, COD)
- ✅ Order success screen with generated Order ID
- 🎨 Smooth animations, floating shoe, marquee banner
- 📱 Custom cursor and scroll reveal effects
